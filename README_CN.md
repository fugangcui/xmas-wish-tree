# 🎄 许愿圣诞树

一个互动的圣诞许愿树网页，让你在平安夜写下愿望，挂在圣诞树上。

🌐 **在线体验**: [xmas.thebesttools.net](https://xmas.thebesttools.net)

📖 **[English Documentation](./README.md)**

![Christmas Tree Preview](https://img.shields.io/badge/🎄-Merry%20Christmas-red?style=for-the-badge)

---

## ✨ 功能特性

| 功能 | 描述 |
|------|------|
| 🌨️ 雪花飘落 | 持续飘落的雪花动画 |
| 🎄 圣诞树 | 精美的多层圣诞树，带发光效果 |
| ⭐ 闪亮星星 | 树顶闪烁的星星 |
| 💡 彩灯效果 | 红黄蓝绿彩灯闪烁发光 |
| ✨ 许愿功能 | 写下愿望挂在树上成为装饰球 |
| 🔮 装饰球 | 愿望变成彩色装饰球 |
| 💾 本地保存 | 愿望保存在本地，刷新不丢失 |
| 🎆 烟花效果 | 绚丽的烟花庆祝 |
| 🎵 背景音乐 | 圣诞音乐播放，可开关 |
| 🎁 礼物盒 | 树下精美的动态礼物装饰 |
| 📱 响应式 | 支持手机和电脑访问 |

---

## 🎮 使用方法

1. **许愿** - 点击「✨ 许个愿望」按钮，写下你的愿望，选择装饰球颜色，挂到圣诞树上
2. **点亮** - 点击「💡 点亮圣诞树」按钮，观看彩灯逐个亮起
3. **烟花** - 点击「🎆 放烟花」按钮，欣赏绚丽烟花
4. **音乐** - 点击右下角「🎵 音乐」按钮，开启/关闭圣诞音乐
5. **查看愿望** - 鼠标悬停在装饰球上，查看愿望内容

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript** - 交互逻辑
- **Canvas** - 雪花和烟花效果
- **LocalStorage** - 本地数据存储
- **Vercel** - 部署托管

## 🚀 本地运行

```bash
# 克隆项目
git clone https://github.com/pockethunter/xmas.git

# 进入目录
cd xmas

# 启动本地服务器
python3 -m http.server 3000
# 或者
npx serve .

# 访问
open http://localhost:3000
```
